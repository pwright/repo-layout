  • Modify to create your own layout

  1. Fork the https://github.com/pwright/repo-layout repo.

  2. Edit the attributes in the antora/modules/ROOT/pages/index.adoc file.

  3. Run npm i to import packages.

  4. Run make render to create HTML.
